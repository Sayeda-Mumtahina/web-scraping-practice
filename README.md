# Web Scraping Practice

Scrapes the list of the largest US companies by revenue from 
Wikipedia and extracts it into a structured pandas DataFrame.

## Libraries
- `requests` — fetches the webpage HTML
- `BeautifulSoup` — parses and extracts the table
- `pandas` — structures the extracted data

## Skills demonstrated
HTML parsing, tabular data extraction, web scraping basics

## Note
This is a learning exercise built while studying Python 
data extraction techniques.
